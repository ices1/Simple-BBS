### Install 
```javascript
    npm install --save
```

### Run
```javascript
    node app.js
```
### Uses
- 后端： `node.js` 搭配 `Express`
- 数据库： `SQL` 
- 前端： 后端 `pug` 模板渲染
- 服务器环境： `Ubuntu 18.04 x86_64  (vps)`

### Feature  
 - 正常登录，注册，发帖，回复等功能
 - 动态验证码登录
 - 邮件找回密码
